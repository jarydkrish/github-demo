# github-demo

> **Note:** This is a demo project for learning purposes.

A simple Node.js console app that multiplies two numbers. This demo demonstrates basic command-line argument parsing and arithmetic operations in Node.js.

## Usage

Run the app with two numbers as command-line arguments:

```bash
node index.js <number1> <number2>
```

### Examples

```bash
# Multiply two positive integers
node index.js 5 3
# Output: 5 * 3 = 15

# Multiply decimal numbers
node index.js 10.5 2
# Output: 10.5 * 2 = 21

# Multiply negative numbers
node index.js -5 10
# Output: -5 * 10 = 50
```

## Requirements

- Node.js (any recent version)

## Installation

No dependencies required. Simply clone and run:

```bash
git clone https://github.com/jarydkrish/github-demo.git
cd github-demo
node index.js 5 3
```
